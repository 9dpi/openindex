# Contributing to OpenIndex Registry

Thank you for contributing to OpenIndex.

This registry prioritizes **accuracy, evidence, and safety** over speed or popularity.

---

## What Can Be Submitted

- Open-source AI systems
- Research models or frameworks
- Datasets or analytic infrastructure

Submissions must relate to **high-stakes domains** (e.g. Health, Finance).

---

## How to Submit a Record

1. Fork this repository
2. Create a new YAML file in the appropriate domain folder
3. Follow the canonical schema exactly
4. Open a Pull Request

---

## Review Criteria

A submission may be **rejected or flagged** if:

- Evidence is missing or misleading
- Claims exceed documented validation
- Regulatory or safety context is unclear
- The description is promotional or speculative

---

## What Reviewers Check

- Schema compliance
- Evidence references
- Intended use clarity
- Risk and regulatory flags

Reviewers do **not** judge performance or outcomes.

---

## Updating an Existing Record

- Do not overwrite history
- All updates must be made via new commits
- Clearly explain the reason for the update in the PR

---

## Code of Conduct

All contributors are expected to act professionally and respectfully.

Disputes are resolved based on **documented evidence**.
